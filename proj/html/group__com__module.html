Detailed Description

Higher level module that handles with the UART and receives/transfers drawing commands

Function Documentation

int command_handler ( )

Handles command receiving.

This function is responsible for checking if new data is coming from another computer and process it. It's called frequently in runDevices() and handles data coming from another computer, analyzing and separating drawing commands, and calling their handlers.

Returns
1 if there is not data available or an error occured, 0 if not
