Detailed Description

Higher level module that handles with mouse/keyboard interrupts and updates their status, according to the current video mode

Function Documentation

int getKeyboard2ByteCode ( )

Returns info about the number of number of make/break codes that the current key has.

If the user presses/releases an arrow key for example (2 byte code), this flag is put to 1. If the user presses/releases a normal key (1 byte code), this flag is put to 0.

int updateKeyboardStatus ( )

Updates the keyboard status in the program.

This function is the top layer of the keyboard module. It's called everytime a keyboard interrupt occurs, and using the interrupt handler to know more info about the code received, fills the static variables "last_key_code", "two_byte_flag" and "press_key_flag".

Returns
1 if a new valid key was received and if we can extract the static variables mentioned above to know more about the key pressed, 0 if not (still processing the interrupts)

int updateMouseStatus ( )

Updates the mouse status in the program.

This function is the top layer of the mouse module. It's called everytime a mouse interrupt occurs, and using the interrupt handler to know more info about the mouse movement or key pressing received, fills the static variables "x_position", "y_position", "RB_pressed", "MB_pressed" and "LB_pressed".

Returns
1 if a new valid finished packet was received and if we can extract the static variables mentioned above to know more about the mouse current state, 0 if not (still processing the interrupts)
